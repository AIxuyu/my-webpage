🛠️ 保姆级安装教程

跟着做，绝对不出错！哪怕你是第一次打开终端也没关系。

觉得图文太长？看视频吧！
无法播放？点击直接前往 B 站观看原视频
1

准备基础环境

别怕，我们只需要装一个简单的运行环境。请前往官网下载并安装 Node.js (推荐下载 LTS 版本)。安装过程一直点“下一步”即可。

2

一键下载 Open Claw

打开你的终端（Windows 叫 cmd 或 PowerShell，Mac 叫终端），复制下面这行神奇的代码并按下回车： 

git clone https://github.com/your-repo/open-claw.git
3

安装依赖并启动

进入文件夹，让它自己把需要的零件组装好，然后就可以起飞啦！依次输入以下命令： 

cd open-claw
npm install
npm start

[替换为运行成功后满是图片的文件夹截图]

一键导出网页高清美图

需求场景

看到喜欢的画师主页，想把几十张高清大图一次性全存下来？一张张右键保存太耗时了。

操作步骤

打开 Open Claw 面板 → 输入目标网址 → 勾选“图片模式” → 点击“开始执行”。

预期效果

去泡杯茶。回来后，所有原图已经自动分类、重命名并保存在你的 `Downloads` 文件夹中了。

[替换为生成的精美 Excel 表格截图]

自动整理每日资讯表格

需求场景

每天需要收集特定关键词的新闻或竞品帖子，手动复制粘贴、排版简直让人崩溃？

操作步骤

在配置文件中输入“关键词” → 设置定时任务（例如每天早上 8 点运行）。

预期效果

每天起床打开电脑，一份包含标题、链接、摘要且排版整齐的 Excel 表格就已经躺在你的桌面上了。
